### get_date_index

## 描述
返回发布文章月份日期的索引

## 请求
方式：http get

url：/api/get_date_index

Content-Type : application/json

## 必选参数
    {
    	无
    } 

## 可选参数
    {
        无
    } 

## 响应
    Content-Type : application/json
    成功返回
    ```
        {status: "ok",…}
        permalinks:["https://www.haoht123.com/date/2019/12/", "https://www.haoht123.com/date/2019/11/",…]
        status:"ok"
        tree:{,…}
    ```
    失败返回
    ```
        无
    ```
